# Release 4.4.39 + Start 4.4.40 (2026-01-27)

This handoff records the 4.4.39 release and sets the baseline for 4.4.40 work.

## What shipped in 4.4.39 (high level)

### Router / IBKR performance (production path)
- Backtesting router for IBKR **futures / cont_future / crypto** now **prefetches the full backtest window once per series** and then returns history via **in-memory slicing**, avoiding `ibkr/iserver/marketdata/history` calls in the hot loop.
- This removes the “one history fetch per simulated minute” failure mode and enables warm-cache backtests to complete in seconds for common 1-week windows.

### Indicators robustness
- `plot_indicators()` hovertext generation no longer crashes when `detail_text` is missing / `NaN` / `pd.NA`.
